# DancingBeans

[DancingBeans] Dev1.0 

## Environment
- API       : [naver]
- Language  : [Swift]
- Tool      : [xcode]


## MVP
- 메뉴 set 
- 메뉴 장바구니
- 메뉴 주문 

## Scenario

1. 주문할 메뉴 클릭
2. 메뉴별 옵션 선정
3. 장바구니에 담기 || 결제
4. 추가 메뉴 클릭 및 선정
5. 장바구니에서 결제
